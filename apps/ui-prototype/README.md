# UI Prototype

React/Vite front-end for SD Agent Studio. The generate screen is wired to the local backend and A1111 WebUI.

Open:

```text
http://127.0.0.1:5177
```

Screens:

- `?screen=overview`
- `?screen=generate`
- `?screen=edit`
- `?screen=lora`
- `?screen=models`
- `?screen=queue`
- `?screen=settings`

Run the Vite dev server:

```bash
npm run dev
```
